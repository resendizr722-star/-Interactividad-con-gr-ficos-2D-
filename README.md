# 🎮 Interactividad con gráficos 2D

Aplicación web interactiva desarrollada con **HTML, CSS y JavaScript**, que simula un pequeño videojuego en canvas donde el usuario debe eliminar objetos en movimiento.

---

## 🚀 Descripción

Este proyecto implementa un sistema de animación en 2D utilizando el elemento `<canvas>`, donde múltiples círculos se desplazan desde la parte inferior hacia la parte superior de la pantalla.

El usuario interactúa con ellos mediante el mouse, eliminándolos antes de que salgan del área visible.

---

## 🧠 Características principales

- 🎯 Detección de colisión con el cursor
- 🔴 Cambio de color al pasar el mouse sobre los objetos
- 🖱️ Eliminación con un solo clic
- 🌫️ Animación de desaparición suave (fade out)
- 📈 Sistema de niveles progresivos
- ⚡ Incremento de velocidad por nivel
- 🚫 Evita superposición de objetos
- 🎮 Pantalla de **Game Over dentro del canvas**
- 🔄 Reinicio del juego con icono ↻
- 📊 Contador de eliminados y progreso (%)
- 📱 Diseño responsive (adaptable)
